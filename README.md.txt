# Sales & Revenue Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard used to analyze sales and revenue performance.

## Tools Used
- Power BI
- Excel

## Dashboard Features
- KPI Cards
- Revenue Trend Analysis
- Regional Sales Analysis
- Payment Method Insights
- Salesperson Performance
- Interactive Filters & Slicers

## Outcome
Developed a business intelligence dashboard to generate insights from sales data using Power BI.
